# 获取中国移动 PT924(G) 等光猫超级管理员密码

## 0x01 背景

移动光猫背面的默认普通用户账号（通常是 `user`）权限极低，无法修改桥接模式、关闭 TR069 远程监控或设置高级防火墙。通过以下方法，可以获取具有完整管理权限的 `CMCCAdmin` 账号密码。

> **注意：** 本方法主要在 **PT924** 和 **PT924G** 型号上测试通过。操作前请务必备份原始配置，以免导致宽带无法上网。

---

## 0x02 准备工作

### 1. 开启光猫 Telnet 隐藏开关

在浏览器地址栏访问以下特殊接口：`http://192.168.1.1/cgi-bin/abcdidfope94e0934jiewru8ew414.cgi`

* **成功标志**：页面显示 `success` 字样。

### 2. 启用本地 Telnet 客户端

由于你使用的是 **WSL/Ubuntu**，建议直接在 WSL 终端操作，或者在 Windows 中开启：

* **Windows**: `控制面板` -> `程序与功能` -> `启用或关闭 Windows 功能` -> 勾选 `Telnet 客户端`。
* **WSL (Ubuntu)**: 执行 `sudo apt install telnet`。

---

## 0x03 攻击实施步骤

### 第一步：连接光猫终端

打开终端（CMD 或 WSL），输入：

```bash
telnet 192.168.1.1
```

根据提示输入后台管理凭据（**注意：输入密码时屏幕不会显示**）：

* **Login**: `yhtcAdmin`
* **Password**: `Cm1@YHfw`

当出现 `#` 符号时，说明你已成功获取光猫系统的 Shell 权限。

### 第二步：定位配置文件

依次执行以下指令进入配置目录并查看关键文件：

```bash
cd /usr/local/ct
vi lastgood.xml
```

### 第三步：搜索管理员密码

进入 `vi` 查看模式后，直接输入以下指令（包含斜杠）并按回车：

```text
/SUSER_PASSWORD
```

此时编辑器会自动跳转到包含密码的行，格式通常如下：

> `SUSER_PASSWORD" Value="******"`

其中 `Value` 后面的内容即为你的 **超级管理员密码**。

> **💡 Vi 操作小贴士：** 获取密码后，输入 `:q!` 然后按回车即可强制退出编辑器而不保存任何更改。

## 0x04 登录验证

1. 重新打开浏览器，访问 `192.168.1.1`。
2. **用户名**：`CMCCAdmin`
3. **密码**：填写你刚才获取到的字符串。

登录后，你将拥有该光猫的最高管理权限。
